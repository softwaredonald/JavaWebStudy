pop和push组成栈的结构：先进后出

pop()从数组尾部弹出一个元素并返回弹出的元素，push从数组尾部压入一个元素，并返回数组的新长度

shift and unshift队列结构：先进先出

从数组的头部弹出一个元素，并返回次元素，索引为0，unshift从头部塞入一个元素 返回数组新长度

var t=[]
t.push(1)
t.push(2)
t.push(3)
console.log(t.pop())
console.log(t.shift())
console.log(t.unshift(4))

sort() 排序是按照字符串方式进行比较，数字会改成字符串 所以数值排序要按照下面算

函数式编程

sort + 函数式编程

        要求：参数接收两个参数，要求返回值为：负数，0，正数。如果放回负数代表第一个参数小于第二个参数，0位相等，正数则大于。
        var comparefun= function(a,b){
            return a-b;
        };
        var m=[51,3,4,5,12,6];
        console.log(m);
        m.sort(m);
        console.log(m);
        利用函数式编程：
        m.sore(comparefun)


        m.sort(function(a,b){return a-b;});
